Pechy32 Insurance Application

Tato aplikace vznikla jako samostatný závěrečný projekt v rámci rekvalifikačního kurzu programátor WWW aplikací naitnetwork.cz

Aplikace simuluje zjednodušenou správu pojišťovny a slouží pouze pro prezentační účely. Data v ní evidovaná jsou zcela fiktivní a nereprezentují skutečné osoby ani pojištění.

Highlights:

  • Aplikace obsahuje kompletní správu (CRUD) pojištěných včetně jejich pojištění
  • Všechna data aplikace jsou uložena v databázi MySQL
  • Logika aplikace (beckendová část) ja napsaná v Javě
  • Aplikace využívá framework Spring Boot Starter Web a šablonovací systém Thymeleaf
  • Aplikace je zabezpečena pomocí Spring Security a rozlišuje uživatele, administrátory a anonymní návštěvníky
  • Aplikace generuje statistiku na základě aktuálního stavu databáze
  • Aplikace je psaná tzv. 'mobile-first' přístupem (upřednosňuje zobrazení na mobilním zařízení)

  • Autor: Tomáš Pecháček
